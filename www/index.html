DirSynch

Introduction

DirSynch is a bi-directional directory synchronizing software.

It helps you to discover the status of each file in both directories: if it's only in one of them, if it's newer in one than in the other or if they are simply equals.

After leting you to choose which files do you want to synchronize, it performs the right operations to make both dirs equals (or not, if you want it different).

It even don't require you to trust it, because you can order him to create a backup of the files before overwriting it.

This project is hosted under CEJUG in Java.Net.
