Tech details

DirSynch is a bi-directional directory synchronizing software.

It helps you to discover the status of each file in both directories: if it's only in one of them, if it's newer in one than in the other or if they are simply equals.

After letting you to choose which files do you want to synchronize, it performs the right operations to make both dirs equals (or not, if you want it different).

It doesn't even require you to trust it, because you can order him to create a backup of the files before overwriting it.

This project is hosted under CEJUG in Java.Net.

If you're a developer, maybe the DirSynch source can be useful to you. Look at the things you can find in it:

  • MD5 hash calculation;
  • Disk access, both reading file's data and listing files in a directory;
  • Recursive function to read the directory tree;
  • Swing interface;
  • Parameters from command line and from ".properties" file;
  • Read and save options in a ".properties" file without needing to restart the program;
  • Show help from an HTML file in a window;
  • Regular expressions evaluation;
  • Use of threads for long operations, with progress bar and cancel option;
  • And other things I can't remember now...
